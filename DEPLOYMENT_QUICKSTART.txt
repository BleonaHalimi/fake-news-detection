TROUBLESHOOTING
================================================================================

Q: start_app.bat doesn't work?
A: Open PowerShell and run: python -m streamlit run app.py

Q: Can't push to GitHub (files too large)?
A: Models might be >100MB. Use Git LFS:
   git lfs install
   git lfs track "models/*.pkl"
   git add .gitattributes
   git commit -m "Track large files"

Q: Deployment fails?
A: Check deployment logs. Usually missing dependencies in requirements.txt

Q: App is slow on free hosting?
A: Normal! Free tiers have limited resources. Use paid hosting for speed.
