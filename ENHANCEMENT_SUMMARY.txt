HOW TO USE THE ENHANCED SYSTEM
================================================================================

1. START STREAMLIT APP:
   streamlit run app.py

   - Paste article text (min 50 chars)

     * Tab 3: Feature Importance (global patterns)

   Time: ~15 minutes

   Output: Optimized model parameters
   Time: 30-60 minutes (can run overnight)

5. TEST ALL FEATURES:
   python test_enhancements.py

   Output: Verification that everything works
   Time: ~30 seconds
